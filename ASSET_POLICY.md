# Visual Asset Policy

The MIT License in this repository applies to the project source code and documentation unless a file states otherwise.

It does not automatically grant rights to:

- customer photographs, portraits, signatures, logos, trademarks, or brand materials;
- commissioned or paid visual designs;
- generated images whose license or source imposes separate conditions;
- third-party fonts, icons, screenshots, promotional composites, or reference images.

Customer-specific assets must not be committed to this public repository. Keep them in an ignored `customer-assets/`, `private/`, or external customer workspace and confirm the necessary rights before running `create-client-package.ps1 -ConfirmAssetRights`.

Public examples must use assets that are original, permissively licensed, or explicitly authorized. Document the source and license beside every example asset.
